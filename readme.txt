درگاه پرداخت جامع، ایمن و هوشمند وردپرس برای تمامی درگاه‌های پرداخت

== Description ==
درگاه پرداخت جامع، ایمن و هوشمند گیت لند، برای اتصال تمامی افزونه‌های وردپرس به تمامی درگاه‌های پرداخت

گیت لند از ۱۱ افزونه وردپرس و ۳۶ درگاه پرداخت پشتیبانی می‌کند، یعنی شما می‌توانید گیت لند را به جای ۳۹۶ افزونه درگاه پرداخت استفاده کنید!

افزونه گیت‌ لند قابلیت سوییچینگ هوشمند بین درگاه‌ها را دارد. این قابلیت به شما کمک می‌کند کمترین نرخ پرش مشتری و بیشترین نرخ پرداخت موفق را داشته باشید. نگران نباشید، گیت‌ لند برای بیش از ۲۰۰ میلیارد تومان و چند هزار تراکنش امتحان خود را پس داده است.

== Installation ==
= نصب =
1. فایل های افزونه را در مسیر `/wp-content/plugins/gateland` آپلود کنید، یا از صفحه افزونه های وردپرس افزونه را مستقیم نصب کنید
1. افزونه را از منو افزونه ها فعال کنید
1. از منو گیت لند > درگاه‌ها، درگاه پرداخت خود را اضافه و پیکربندی کنید
1. با قدرت به فروش خود آماده دهید!
